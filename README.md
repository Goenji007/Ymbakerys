# YM Bakery's - Website Redesign

Premium website redesign for YM Bakery's, a gourmet artisanal bakery.

## Project Structure
The project is organized in a clean and optimal professional structure:
- `/assets/img/`: Contains all optimized brand and product images.
- `/assets/css/`: Legacy styles and custom CSS definitions.
- `Index.html`: The main entry point using a modern Tailwind CSS architecture with dark mode support.
- `README.md`: This file.

## Features
- **Modern UI/UX**: Clean, readable, and premium aesthetic based on professional bakery standards.
- **Dark Mode**: Fully implemented theme switching for accessibility and style.
- **Responsive Branding**: Optimized for all devices from mobile to large screens.
- **Direct WhatsApp Integration**: Fast and easy ordering process.

## How to Run locally
1. Use a local server (e.g., `npx http-server -p 8080`).
2. Open `http://localhost:8080` in your browser.
